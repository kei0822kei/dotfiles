-- file --

I;    open or close dot file  
r;    update  


## braceless
zc; カーソルの下の折畳を一段階閉じる  
zC; カーソルの下の折畳を再帰的に全て閉じる  
zo; カーソルの下の折畳を一段階開く  
zO; カーソルの下の折畳を再帰的に全て開く  
zM; 全ての折畳を閉じる（前に操作した対象のみ）  
zR; 全ての折畳を開く
